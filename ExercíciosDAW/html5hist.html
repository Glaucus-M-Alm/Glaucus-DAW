A história do HTML5

  • Com o surgimento da web, era necessário criar uma linguagem que fosse entendida por meios de acesso diferentes. Para tanto, Tim Berners-Lee desenvolveu o HTML
  • HTML é uma abreviação de Hypertext Markup Language, ou seja, Linguagem de Marcação de Hipertexto. Resumindo, o HTML é uma linguagem usada para a publicação de conteúdo (texto, imagens, vídeos, áudio etc.) na web.
  • Entre 1993 e 1995, surgiram novas versões (HTML+, HTML 2.0, HTML 3.0) onde foram propostas diversas mudanças para enriquecer as possibilidades da linguagem
  • O HTML5 é a nova versão do HTML4 fazendo com que isso fique transparente para o usuário final.
  • um dos seus principais objetivos é facilitar a manipulação dos elementos, possibilitando o desenvolvedor modificar as características dos objetos de forma não intrusiva,
  • fazendo com que isso fique transparente para o usuário final.
  • diferente das versões anteriores, o HTML5 fornece ferramentas para o CSS e o Javascript fazerem seu trabalho da melhor forma possível
  • Houve também modificações na forma em que escrevemos o código e organizamos a página. Ela passou a ser mais semântica com menos códigos

Diferenças entre XHTML e HTML

  • XHTML é mais estrito que o HTML
  • Pelo fato de o html ser menos estrito, ele é adotado como padrão, pois é mais possível de mais pessoas utilizarem uma linguagem mais simples do que uma restrita
